    Afinal que é HTML?

    (voltar)

    Já vou começar dizendo que HTML não uma linguagem de programação como muitos pensam e sim uma linguagem de marcação.

    Com o HTML é possível criar vários tipos de elementos que compõem um site e tudo do jeito que você quiser. Podendo escolher fontes de texto, cor, imagens e muitas outras coisas que deixaram o site muito atrativo visualmente.

    O HTML é formado, principalmente, pelas tags. Inclusive é sobre o que falaremos a seguir.

    O que são Tags?

    (voltar)

    As tags são palavras chaves que servem para organizar conteúdos.

    Dentro de um site cada pequena frase, imagem, link tem uma tag por trás para que funcione da forma que você está vendo.

    Sabe quando você tem que fazer uma pasta de arquivos e então organiza cada uma delas para que a pessoa que vai receber aqueles arquivos consiga navegar e encontrar tudo que em ali dentro sem problemas? É exatamente isso que as tags fazer, só que por trás das câmeras onde só quem consegue vê-las é o desenvolvedor.

    As tags mais usadas

    (voltar)

    A primeira tag, que é a que abre um novo projeto, é a tag HTML. E dentro dessa tag todas as outras são inseridas possibilitando a existência do site.

    Outras tags fundamentais são Head e Body que significam cabeça e corpo.

    Tudo que está dentro da tag Head é o que carrega antes de ser exibido ao usuário e tudo que esta na tag Body é o que o usuário de fato vê. A tag Body é o que faz a magia acontecer. Tudo que você vê num site só existe por causa dela e por isso tem esse nome, afinal nada se sustenta sem um corpo.

    Tags básicas como I e Strong também são bastante utilizadas e basicamente serve para colocar palavras em itálico e negrito.

    Umas das que acho mais legais é a tag Style. Não! Não estamos falando de Harry Styles (trocadilho bem ruim aliás.)

    A tag Style serve para deixar o texto mais "estiloso”. Com ele você pode modificar as cores do texto por exemplo. Mas esse é mais voltado pro CSS e não estamos focados nessa parte ainda.

    Também temos as tags h que vão de h1 a h6 e representam 6 níveis de seleção de tamanho de texto sendo o h1 o mais alto e o h6 o mais baixo.

    A tag img serve para adicionar uma imagem ao site.

    Existem muitas tags usadas para criar uma pagina web. Essas são apenas alguns exemplos. Há uma lista enorme de tags que tornam nossa experiencia navegando por sites muito mais satisfatória.
